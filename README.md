# Continuum – Cultural Memory Preservation Platform

Continuum is a human-centered digital platform designed to preserve cultural memories such as languages, traditions, places, food practices, and personal experiences.

The platform allows users to document meaningful cultural stories so that knowledge and identity can continue across generations.

## Features (Frontend Implemented)

Profile Page
- Displays user information
- Shows personal note and known languages
- Sections for My Stories and Saved Stories

Explore Stories
- Browse cultural stories shared by users
- Story cards display title, image, place, language, and description
- Grid/List viewing options

Story View
- Displays story content with metadata
- Includes place, language, category, and year

My Stories / Saved Stories
- View stories created by the user
- View saved stories
